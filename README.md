# CoffeeSeller
Програмний код до дослідницької роботи на тему "LLM як можлива альтернатива програмісту". Сгенерован LLM на html/js виключно за допомогою промптів, вказаних у роботі.

## Структура проєкту:
drinks.html - головний файл проекту<br>
coffee-data-backup.json - файл з даними до проекту

## Як запустити проєкт:
1. Завантажити **html**-файл та **json**-файл на локальний диск.
2. Відкрити **html**-файл у Chrome.
3. Для завантаженя даних у програму натиснути кнопку "Вибрати файл" внизу вікна програми та обрати **json**-файл, завантажений раніше.
